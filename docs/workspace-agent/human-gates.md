# Human Gates

Human approval is a production control, not a courtesy.

## Required Gates

- route approval
- source claim approval
- chart proof approval
- asset board approval
- Lottie candidate approval
- styleframe approval
- Codex readiness approval
- still/contact-sheet approval
- draft MP4 approval
- final render approval

## Gate Failure

If any gate is missing, the packet stack remains blocked.

No production reel implementation starts in Phase 1.
