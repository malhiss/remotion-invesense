# Lottie Research Playbook

LottieFiles is an asset source, not a creative director.

## Search By Motion Role

Search for the job the asset must perform:

- route pulse
- highlight sweep
- chart accent
- payment transfer
- data transfer
- checkmark
- warning x
- loader
- proof burst
- CTA pulse

Do not search only for generic finance taste.

## Candidate Requirements

Every candidate must record:

- source URL
- creator or source page
- license note
- preview behavior
- approved usage role
- what to use
- what not to copy
- fallback native Remotion option

## Approval Rule

Codex downloads only approved Lottie JSON or uses a manually supplied local file.
Lottie cannot be the hero metaphor. It supports route energy, timing texture, proof burst, CTA pulse, or small icon behavior.
