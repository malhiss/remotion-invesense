# Codex Handoff Rules

Codex may reject a handoff if the packet stack is incomplete.

## Required Handoff Contents

- source-locked claims
- chart proof role and recreation mode
- selected route and rejected alternatives
- frame ranges
- hero asset behavior
- proof birth moment
- final residue frame
- asset board
- Lottie candidates or explicit no-Lottie decision
- approved styleframe targets
- allowed packages and gated packages
- forbidden inventions
- render permissions
- validation commands

## Blockers

- text-only handoffs are blocked for complex metaphor reels
- detached proof cards are blocked
- chart wallpaper is blocked
- unapproved Lottie hero metaphors are blocked
- Codex route invention is blocked
- final render approval must be explicit

Codex is implementation-only and may stop if asked to invent creative strategy.
