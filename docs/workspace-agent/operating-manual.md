# Workspace Agent Operating Manual

Workspace Agent is the creative director. Codex is implementation-only.

## Operating Contract

Workspace Agent owns:

- source intake and source boundaries
- evidence audit and source claim locks
- financial mechanism diagnosis
- benchmark matching
- route and analogy selection
- asset board and Lottie search planning
- styleframe direction
- Codex readiness scoring
- Codex handoff drafting

Codex owns:

- deterministic Remotion implementation from an approved handoff
- validation commands
- still/contact-sheet review exports
- reporting implementation risks

## Hard Separations

- Codex does not choose the financial route.
- Codex does not invent source numbers or claims.
- Codex does not invent the visual style.
- Codex does not replace approved benchmark direction.
- Codex is implementation-only after Workspace Agent creates an approved packet stack.

## Phase 1 Rule

Phase 1 builds the knowledge layer only. No production reel implementation has started.
