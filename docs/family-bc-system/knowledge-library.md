# Family B/C Knowledge Library

This library rewrites useful old research into new Family B/C rules. It is planning guidance, not migrated implementation.

## Core Motifs

- Chart as terrain: a chart line, moving average, candlestick, bar, axis, or marker becomes the world where the event happens.
- Support/resistance as physical line: a source line catches, bends, breaks, or fails while preserving chart honesty.
- Strict bars as measured objects: bars are data graphics, not progress UI; values sit on or inside the bars.
- Logo as actor: corporate, asset, fund, bank, exchange, or protocol logos behave as participants in the mechanism.
- Route/rail: money, policy, tokens, liquidity, or ownership moves through visible paths and endpoints.
- Stamp/latch: proof locks only after settlement, approval, launch, certification, or source event.
- Scanner/filter: useful for Shariah, quality, eligibility, compliance, index inclusion, and screening only when contact causes pass or reject.
- Gap/bracket/caliper: useful for spread, valuation gap, retained-vs-lost, earnings-vs-price, yield gap, or cost drag.
- Drain/leak/siphon: useful for fees, inflation, taxes, leakage, decay, hidden drag, or outflows.
- Vault/unlock/tokenization: useful for custody, access, private markets, settlement rails, RWA, and tokenization.
- CTA residue: final CTA inherits a chart line, underline, stamp, bracket, route, scar, or marker instead of becoming a generic card.

## Wow Asset Events

- Line Bend + Snap: resistance, threshold failure, support break.
- Floor Crack / Fall-Through: support failure and drawdown.
- Scanner Reveal: hidden criteria, compliance, source inspection.
- Scanner Reject Path: rejected exposure changes color only after contact.
- Gate Open / Close: inclusion, threshold, access, eligibility.
- Tank Fill / Drain Mark: liquidity, reserves, outflows, fee drag.
- Pipe Bottleneck: settlement friction, liquidity bottleneck, throughput limit.
- Tether Stretch / Pullback: mean reversion, spread pullback, valuation snapback.
- Clamp Compression: valuation compression, rate pressure, margin squeeze.
- Ledger Stamp: settlement, official proof, source-locked conclusion.
- Magnifier Reveal: document proof, inspection, hidden source detail.
- Route Endpoint: payment rails, token movement, policy transmission.
- Proof Latch: evidence clicks into the mechanism.
- Caliper Spread: gap measurement, valuation spread, yield spread.
- Vault Unlock: access, custody, tokenization, private-market opening.

## Rejection Rules

- Reject chart-only scenes if the chart does not behave or reveal proof.
- Reject generic progress bars.
- Reject pasted proof cards.
- Reject detached CTA cards.
- Reject Lottie as decorative clutter.
- Reject labels that exist because the asset failed.
- Reject any proof that appears before the event that creates it.
