# Workspace Agent Creative Director Workflow

Workspace Agent is the creative director. Codex is the implementation specialist.

## Operating Split

Workspace Agent owns source intake, evidence audit, mechanism diagnosis, benchmark matching, analogy selection, asset planning, Lottie search planning, visual reference board, styleframe direction, and Codex handoff.

Codex owns deterministic Remotion implementation from approved handoff only. Codex does not invent claims, route logic, financial analogy, style direction, or source numbers.

## Packet Order

1. Source intake packet.
2. Evidence audit packet.
3. Mechanism diagnosis packet.
4. Benchmark deconstruction packet.
5. Analogy route packet.
6. Chart proof packet.
7. Asset board packet.
8. Lottie search packet.
9. Styleframe direction packet.
10. Codex readiness scorecard.
11. Codex handoff packet.

## Human Gates

- Route approval.
- Source claim approval.
- Chart proof approval.
- Asset and Lottie approval.
- Styleframe approval.
- Codex readiness approval.
- Contact-sheet approval.
- Draft MP4 approval.
- Final render approval.

## Codex Rejection Rights

Codex may reject a handoff if it lacks source-locked claims, chart proof role, asset decisions, Lottie license notes, approved visual targets, frame ranges, allowed packages, or render permissions.

## Non-Template Rule

The system provides contracts, packets, primitives, and QA gates. It must not become a repeatable template generator. Each reel still needs a custom route, custom proof hierarchy, custom asset plan, and human-approved visual target.
